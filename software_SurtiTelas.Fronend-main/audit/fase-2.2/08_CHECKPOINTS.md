# 08_CHECKPOINTS.md

## Checkpoints obligatorios

| ID | Checkpoint | Cuándo |
|----|------------|--------|
| CP-01 | `npm run build` | Antes de iniciar Fase 3 (baseline) |
| CP-02 | `npm run lint` | Después de mover providers |
| CP-03 | `npm run build` | Después de cada wave |
| CP-04 | `npm run dev` + navegación manual | Después de routing |
| CP-05 | Revisión de imports rotos | Después de UI consolidation |
| CP-06 | `npm run test` (si existe) | Antes de eliminar duplicados |
| CP-07 | Validación runtime en staging | Antes de merge a main |
